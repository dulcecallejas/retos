## Array sum

Given an array of integers, find the sum of its elements.

Run the tests in the editor to see the result.

### Constrains:
- ar[i] <= 1000
- `Complete the given code` to solve it
- Can not use addition built in functions

### Input example
[1,2,3]

### Output example
6

### Explanation
1 + 2 + 3 = 6
